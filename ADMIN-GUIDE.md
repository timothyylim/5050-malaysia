# 50-50 Malaysia editor guide

Use the editor at `https://5050malaysia.com/admin/`.

## Add an expert

1. Open **Experts** and choose **New Expert**.
2. Enter the expert's name and choose one or more industries.
3. Add the role, organisation, specialisation, email, and phone where available.
4. Leave **Hide from public directory** off for a public listing.
5. Click **Publish**.

## Edit an expert

Open **Experts**, select the profile, make the change, and click **Publish**. Leave
**Public URL slug** unchanged when editing an existing profile; it is the profile's web
address.

Industry counts are calculated automatically from the expert profiles. There is no count
to maintain manually.

## Edit About or FAQs

Open **Pages**, choose **About** or **FAQs**, make the change, and click **Publish**.

The first sign-in uses GitHub authorization. After that, all editing happens in this
editor; there is no need to visit GitHub or edit files.
